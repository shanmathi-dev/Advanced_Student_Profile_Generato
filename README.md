# 🎓 Advanced Student Profile Generator

An advanced web-based Student Profile Generator developed using HTML, CSS and JavaScript. This application allows students to create, manage and save professional profile cards with dynamic statistics and personalized information.

## 🚀 Features

- Generate Student Profile
- Upload Student Photo
- Dynamic Student ID Generation
- State and City Selection
- Search Student Profile
- Save Profile
- Reset Profile
- Clear All Data
- Dark / Light Mode
- Copy Student ID
- Name Length Calculator
- Vowel Counter
- Profile Score Calculation
- Responsive Design
- Local Storage Integration
